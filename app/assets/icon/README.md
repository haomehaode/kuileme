# APP图标说明

请将APP图标文件放在此目录下，文件名为：`app_icon.png`

## 图标要求：
- 格式：PNG
- 尺寸：建议 1024x1024 像素（正方形）
- 背景：透明或深色背景
- 内容：包含绿色下降趋势图标的图标

## 使用说明：
1. 将图标文件命名为 `app_icon.png` 并放在此目录
2. 运行以下命令生成所有平台的图标：
   ```bash
   flutter pub get
   dart run flutter_launcher_icons
   ```
